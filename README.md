# Echoes

一款帮助专注的 iOS App：用可自由叠加的白噪音营造氛围，再用 AI 把一件大事拆成马上能开始的第一步。这个仓库是它的可交互网页 demo，请用电脑打开体验。

**体验链接：https://echoes-ai.pages.dev/**

## 解决什么问题

用户面对又大又笼统的任务时，常常不知道第一步该做什么，于是一直拖着不开始。卡住的往往不是能力，而是任务太模糊、没有明确的入口。

另一方面，不少人需要一点背景声才能进入状态，完全安静反而坐不住。但市面上的白噪音产品大多只给固定几条音轨，没法按场景自由组合出想要的氛围。

## 产品怎么做

打开后是一台可以真实点、拖的手机原型，主要做两件事。

一是白噪音混音。声音按雨、火、风等分类，可以自由叠加、各自调音量，组合出「雨声加远处的火再加一点风」这样的层次，常用的组合可以存成预设。

二是 AI 任务拆解。用户输入一件想做的事，AI 先问一两个关键问题把需求问清楚，再拆成几步动词开头、能照着做的清单；上到主屏后一次只盯一步，完成一步自动推进。

## 关键设计

- 功能做减法。砍掉了冥想、日志等设想，只留白噪音和任务拆解两条主线，因为它们正好串成「先营造氛围，再开始专注」一条完整的路径，其余容易分散注意力。
- AI 只用来帮用户开始。它不替用户做决定、不自动改设置，只把「不知道怎么下手」变成「第一步很清楚」，而且给的永远是可以改的草稿。
- 先问清楚，再拆解。同一件事，目的不同，拆法完全不同，所以 AI 会先确认再给步骤，而不是上来就拆出一份看着对、其实用不上的清单。
- 沉浸模式不一刀切全屏。界面淡出后，用户可以自己选择留下时钟、播放条还是混音台，拼出想要的极简程度。
- 专注计时和氛围联动，倒计时归零自动收声，让一次专注有始有终。

## 技术

手机原型用 Next.js 和 React 开发，部署在 Cloudflare，AI 用大模型 API。页面上还有一个「问一问」助手，可以回答关于这个项目的问题。
